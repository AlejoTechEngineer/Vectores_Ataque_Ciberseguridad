### рҹҺЈ 01 В· Phishing / Smishing

**ВҝQuГ© es?** TГ©cnica de ingenierГӯa social que suplanta entidades de confianza para robar credenciales. El smishing usa SMS como vector.

**Vector de ataque:**
```
Reconocimiento (OSINT) вҶ’ Clonar sitio web вҶ’ EnvГӯo masivo вҶ’ Captura credenciales вҶ’ ExplotaciГіn
```

**Archivos:**
- `phishing_smishing.py` вҶ’ Servidor Flask que simula captura de credenciales
- `detector_cabeceras.py` вҶ’ AnГЎlisis defensivo de cabeceras SPF/DKIM/DMARC

---

### рҹ”’ 02 В· Ransomware

**ВҝQuГ© es?** Malware que cifra archivos con AES-256 + RSA-2048 y exige rescate en criptomonedas. Casos: WannaCry, REvil, LockBit.

**Vector de ataque:**
```
Phishing inicial вҶ’ Persistencia (registro Windows) вҶ’ Movimiento lateral вҶ’ Cifrado masivo вҶ’ Rescate
```

**Archivos:**
- `ransomware.py` вҶ’ SimulaciГіn del mecanismo de cifrado hГӯbrido
- `detector_ransomware.py` вҶ’ Monitor de comportamiento con watchdog

> [!WARNING]
> El script `ransomware.py` **SOLO debe ejecutarse en una carpeta de sandbox aislada**. Nunca en directorios con archivos reales.

---

### вҢЁпёҸ 03 В· Keylogger

**ВҝQuГ© es?** Software que registra en secreto todas las pulsaciones de teclado y las exfiltra periГіdicamente al atacante.

**Vector de ataque:**
```
DistribuciГіn (adjunto malicioso / software pirata) вҶ’ InstalaciГіn oculta вҶ’ Hook WinAPI вҶ’ ExfiltraciГіn SMTP
```

**Archivos:**
- `keylogger.py` вҶ’ Captura de pulsaciones con pynput + exfiltraciГіn simulada
- `detector_hooks_teclado.py` вҶ’ DetecciГіn de procesos y DLLs sospechosas

---

### рҹӨ– 04 В· Botnet

**ВҝQuГ© es?** Red de equipos zombi controlados por servidor C2. Usada para DDoS, spam, cryptojacking. Caso Mirai: 600.000+ dispositivos IoT.

**Vector de ataque:**
```
InfecciГіn masiva (IoT / malware) вҶ’ ComunicaciГіn C2 (beaconing) вҶ’ EjecuciГіn Гіrdenes вҶ’ Mantenimiento red
```

**Archivos:**
- `botnet_server.py` вҶ’ Servidor C2 con Flask (registro de bots, cola de comandos)
- `botnet_client.py` вҶ’ Cliente bot con check-in periГіdico

---

### рҹ•ҙпёҸ 05 В· Fraude del CEO (BEC)

**ВҝQuГ© es?** SuplantaciГіn del CEO para ordenar transferencias fraudulentas. Sin malware, puro engaГұo psicolГіgico. El FBI reportГі $2.900M en pГ©rdidas en 2023.

**Vector de ataque:**
```
OSINT profundo (semanas) вҶ’ SuplantaciГіn email вҶ’ Urgencia + confidencialidad вҶ’ Transferencia вҶ’ Lavado
```

**Archivos:**
- `fraude_ceo.py` вҶ’ RecopilaciГіn de estructura organizativa pГәblica (OSINT)
- `verificador_spf_dmarc.py` вҶ’ VerificaciГіn de autenticaciГіn de dominios
